# Changelog

## [0.1.4] - 2024-12-14

### Changed
- **BREAKING**: По умолчанию используются только динамические инструменты (3 инструмента) вместо всех статических (126 инструментов)

- `--include-all-tools` теперь `false` по умолчанию
- Обновлена документация с рекомендациями по использованию

- Решена проблема превышения лимита инструментов Cursor (80 инструментов)
