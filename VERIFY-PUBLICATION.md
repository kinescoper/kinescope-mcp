## Если пакет не появляется

1. **Проверьте права доступа:**
   ```bash
   npm access ls-packages
   ```
   Должен показать `@kinescope/mcp: read-write`

2. **Проверьте публичность scoped пакета:**
   Scoped пакеты (`@scope/name`) требуют `--access public` при первой публикации

3. **Проверьте логи:**
   ```bash
   cat ~/.npm/_logs/*-debug-*.log | tail -50
   ```

4. **Попробуйте опубликовать еще раз:**
   ```bash
   npm publish --access public --dry-run  # Проверка
   npm publish --access public            # Публикация
   ```
